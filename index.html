Tentang Kami

Visi & Misi

Visi:

Menjadi sekolah unggulan yang menghasilkan lulusan berkarakter, berprestasi, dan siap menghadapi tantangan global.

Misi:

  • • Menyelenggarakan pendidikan berkualitas tinggi
  • • Mengembangkan karakter siswa yang berakhlak mulia
  • • Memfasilitasi pengembangan bakat dan minat siswa
  • • Mempersiapkan siswa untuk pendidikan tinggi dan karir

Program Akademik

Kami menawarkan berbagai program akademik unggulan untuk mengembangkan potensi siswa

Program IPA

Fokus pada mata pelajaran sains dan matematika untuk persiapan masuk fakultas kedokteran, teknik, dan MIPA.

  • • Matematika Peminatan
  • • Fisika
  • • Kimia
  • • Biologi

Program IPS

Mengembangkan pemahaman tentang ilmu sosial, ekonomi, dan humaniora untuk persiapan fakultas sosial.

  • • Ekonomi
  • • Sosiologi
  • • Geografi
  • • Sejarah Peminatan

Program Bahasa

Mengasah kemampuan berbahasa dan sastra untuk persiapan fakultas bahasa dan komunikasi.

  • • Bahasa Indonesia
  • • Bahasa Inggris
  • • Bahasa Mandarin
  • • Sastra Indonesia

Ekstrakurikuler

Berbagai kegiatan pengembangan bakat dan minat di luar jam pelajaran.

  • • Pramuka
  • • Olahraga
  • • Seni & Budaya
  • • Robotika

Olimpiade Sains

Program persiapan kompetisi sains nasional dan internasional.

  • • OSN Matematika
  • • OSN Fisika
  • • OSN Kimia
  • • OSN Biologi

Kelas Unggulan

Program akselerasi untuk siswa berprestasi dengan kurikulum diperkaya.

  • • Pembelajaran Adaptif
  • • Riset Mandiri
  • • Mentoring Khusus
  • • Persiapan PTN
